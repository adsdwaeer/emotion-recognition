# 😊 Распознавание эмоций на лицах

Веб-приложение на Flask для определения эмоций по фотографии лица с использованием готовой модели Hugging Face.

**Автор:** Кузнецов Никита  
**Курс:** Программная инженерия, АлтГУ 2025–2026  
**Роли:** архитектор · программист · тестировщик · технический писатель · Scrum-мастер

---

## Используемые технологии

| Компонент | Технология |
|---|---|
| ML-модель | [dima806/facial_emotions_image_detection](https://huggingface.co/dima806/facial_emotions_image_detection) |
| Архитектура модели | Vision Transformer (ViT) |
| Библиотека ML | Hugging Face Transformers |
| Веб-фреймворк | Flask |
| Фронтенд | HTML + CSS (без сторонних фреймворков) |

---

## Что умеет приложение

- Принимает фотографию лица (JPG / PNG / WEBP, до 10 МБ)
- Определяет одну из **7 эмоций**: злость, отвращение, страх, радость, нейтрально, грусть, удивление
- Показывает уверенность модели по каждой эмоции в виде столбчатой диаграммы

## Как работает модель

Модель `dima806/facial_emotions_image_detection` основана на архитектуре **Vision Transformer (ViT)**. Она обучена на датасете FER2013+ и показывает точность ~80–85% на тестовой выборке.

Принцип работы:
1. Изображение разбивается на патчи (patches)
2. Каждый патч кодируется в вектор (embedding)
3. Transformer-энкодер обрабатывает последовательность патчей
4. Классификационная голова выдаёт вероятности по 7 классам эмоций
